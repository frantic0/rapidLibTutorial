Getting started with RapidLib for Machine Learning

This page guides you how use RapidLib to build a simple machine learning object and use it, in just a few steps.

1. Add RapidLib to your HTML page 

      
          // Just include this script tag inside your HTML page
          <script src="RapidLib.js"></script>

2. Create a Machine Learning object with RapidLib

RapidLib is available on the Javascript global scope 
        
          var rapidLib = RapidLib();
          var myRegression = new rapidLib.Regression();

3. Create a data set from a Javascript object definition

        
          var myData = [
            {
              input: [48],
              output: [130.81]
            },
            {
              input: [54],
              output: [185.00]
            },
            {
              input: [60],
              output: [261.63]
            },
            { 
              input: [66], 
              output: [369.994] 
            },
            { 
              input: [72], 
              output: [523.25] 
            }
          ];

4. Train a Machine Learning object with a data set 

        
          myRegression.train(myData);

5. Run a trained Machine Learning object on new data

        
          var rapidInput = [64];
          var freqHz = myRegression.run(rapidInput);
          console.log(freqHz);
